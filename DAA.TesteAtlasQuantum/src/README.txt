1. Abra o projeto no Visual Studio.
2. Na aba Server Explorer, clique em Connect to server.
3. No campo Server Name digite "(localdb)\mssqllocaldb".
4. Em Select or enter a database name digite "DAATesteAtlasQuantum" e clique em OK.
5. Ao ser questionado se deseja criar a base de dados informada clique em Yes.
6. Entre no Package Manager Console, e execute o comando "update-database".
