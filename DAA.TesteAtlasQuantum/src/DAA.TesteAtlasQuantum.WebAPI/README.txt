1. Execute o script na instância local (MSSQLLocalDB):
	Server name: (localdb)\mssqllocaldb
	Authentication: Windows Authentication

2. URLs
	 - Lista todas as transações: https://localhost:44337/api/transacoes (GET)
	 - Lista transações por ID : https://localhost:44337/api/transacoes/1 (GET)
	 - Insere transações: https://localhost:44337/api/transacoes (POST)
